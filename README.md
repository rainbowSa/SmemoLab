# SmemoLab
Final project work of the course Java Junior Developer sponsored by Generation Italy, SmemoLab is a Java web-app where you can use virtual post-it.
## technology used
- Java
- JPA
- EclipseLimk
- MySql
- Boostrap
- HTML
- CSS
- JavaScript
